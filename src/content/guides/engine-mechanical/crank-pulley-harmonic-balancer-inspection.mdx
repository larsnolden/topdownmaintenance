## Before You Start / Safety

This guide is for **Mazda MX-5 NB (1998-2005, NB1/NB2)** only.

A failing crank pulley/harmonic balancer can look like accessory-belt noise or front-engine vibration. Diagnose first; replace only when confirmed.

> Warning: Keep fingers, sleeves, and tools away from rotating pulleys and belts; never hand-check pulley tracking with the engine running. https://www.aa1car.com/library/serpentine_belt.htm

> Manual-reference quote: Mazda NB service references are split by year group in the archive (including "1999-2001 Mazda Miata Service Repair Manual" and "2005 Mazda Miata Service Repair Manual"); use the year-correct manual before final torque/reassembly. https://www.mellens.net/mazda/index.html

## Required Tools

- Metric socket/wrench set
- Breaker bar
- Torque wrench (sized for crank-bolt range)
- Crank pulley holding tool (or approved equivalent)
- Puller set suitable for crank pulley/harmonic balancer service
- Inspection light
- Marker/paint pen

## Required Parts / Fluids

- NB-compatible crank pulley / harmonic balancer (if replacement is confirmed)
- Crank bolt/washer if your manual requires replacement or if hardware is damaged
- Accessory belt(s) if worn or contaminated

## Model-specific notes (NB1 vs NB2)

### NB1 (1998-2000)
- Service concept is the same: inspect runout/wobble/noise first, then replace if confirmed.
- Verify front-end hardware sequence and torque process in your year-correct manual.

### NB2 (2001-2005)
- Same service logic, with possible packaging and accessory differences by market/options.
- Do not assume one universal torque/process table across all NB2 variants.

## Step-by-Step Procedure

### 1) Confirm symptoms before disassembly

Look for:
- visible pulley wobble,
- repeat belt throw/shredding,
- chirp/squeal with otherwise correct belt condition,
- front-engine vibration that changes with RPM.

> Community quote: Community search results repeatedly surface NB-specific "crank pulley wobble" reports and replacement questions, indicating this is a known front-engine diagnostic pattern for owners. https://www.reddit.com/r/Miata/search/?q=NB%20crank%20pulley&restrict_sr=1

### 2) Rule out simpler belt issues first

Before condemning the balancer:
- inspect belt condition/tension,
- inspect adjacent pulleys/idlers,
- verify no obvious bracket misalignment.

> High-risk quote: Belt noise/slippage can be caused by tension loss or pulley misalignment; replacing one part without checking alignment can miss root cause. https://www.aa1car.com/library/serpentine_belt.htm

### 3) Prepare for crank-pulley removal

- Disconnect battery if working near starter/cable paths.
- Set safe access and remove belts/components needed for crank pulley access.
- Use a **proper holding method** for crank bolt removal.

> High-risk quote: NB timing-belt front-end procedures call out a dedicated method/tool to hold the crank while loosening/tightening the crankshaft bolt. https://mx5unleashed.com/tech/timing-belt-mx5.html

### 4) Remove and inspect pulley/balancer

- Remove crank bolt with crank held correctly.
- Pull pulley/balancer using the correct puller method.
- Inspect for:
  - visible runout damage,
  - keyway/key damage,
  - separated/deteriorated damper material,
  - contact/rub marks.

### 5) Install replacement (if confirmed faulty)

- Clean mating surfaces.
- Install pulley/balancer squarely (no hammering on unsupported surfaces).
- Reinstall hardware per year/VIN manual process.
- Set final crank-bolt torque and angle/sequence per manual.

> High-risk quote: Front-end timing service instructions repeatedly emphasize torque wrench use and controlled reassembly for crank pulley hardware. https://mx5unleashed.com/tech/timing-belt-mx5.html

### 6) Reassemble and verify

- Refit belts and set tension correctly.
- Start engine and observe pulley tracking.
- Check for vibration, belt wander, and abnormal noise.
- Reinspect after initial heat cycle.

> Community confirmation quote: MX-5 community search results also show repeated front-engine rattle/timing-jump discussions around pulley/timing-area service, reinforcing post-assembly verification before normal driving. https://www.mx5nutz.com/search/2144465/?q=crank+pulley+nb&o=relevance

## Torque Specs / Capacities

- **Crank-bolt torque and any angle sequence:** use year/VIN-specific FSM values.
- **Accessory-belt retension values:** use year/VIN-specific FSM values.

This guide intentionally avoids publishing one universal NB numeric value because publicly accessible sources in this run were inconsistent and often non-OEM.

## Verification / Post-service checks

- Pulley runs true with no visible wobble.
- Belt tracks centered with no rapid edge wear.
- No abnormal front-engine vibration at idle and light throttle.
- Recheck hardware and belt condition after short road test and cool-down.

## Uncertainty / Open Questions

- Publicly accessible OEM NB crank-bolt numeric values were not consistently extractable in this run.
- Several major generic how-to sites were region/cookie blocked during research, so this guide is intentionally conservative and manual-first.
- Confidence is high for process order and diagnostic flow, moderate for publicly available cross-market numeric detail.
