## Before You Start / Safety

- This appendix is a planning reference, not a substitute for year/VIN-specific workshop data.
- Never mix unknown fluid types without confirming compatibility.
- When in doubt, prioritize cap label + owner/workshop manual for your exact car.

> Warning: "On some cars, the power-steering fluid level can only be checked accurately after the engine has run for a brief period..." https://www.wikihow.com/Check-and-Add-Power-Steering-Fluid

## Required Tools

- Owner/workshop manual reference
- Service log sheet
- Clean measuring containers/funnels
- Labels/marker for opened fluids

## Required Parts / Fluids

Typical service-stock items for NB ownership:
- Engine oil (viscosity per climate/use and manual)
- Oil filter + drain washer
- Coolant (correct chemistry for system)
- Brake/clutch hydraulic fluid (per cap/manual)
- Transmission fluid (correct type for gearbox)
- Differential oil (correct type/grade)
- Power steering fluid/ATF-equivalent per manual language

## Model-specific notes (NB1 vs NB2)

- NB fluid requirements vary by engine, gearbox type, market, and prior modifications.
- Published community “baseline” lists are useful starting points, but final fill choices must be validated per vehicle.

## Step-by-Step Procedure

### 1) Build your vehicle-specific fluid profile

Record: year, engine, transmission type, differential type (open/LSD if known), climate, and driving style (street/track).

### 2) Confirm each service fluid from primary labels/manual

For each system, document:
- fluid type/spec wording,
- approximate capacity range,
- interval trigger (time, mileage, or condition).

### 3) Track “service fill” vs “dry fill” separately

Capacities often differ between routine drain/refill and complete dry rebuild fill.

### 4) Log consumable part numbers and brands

Record filters, crush washers, and any chemistry brand used, so future troubleshooting can trace compatibility.

> High-risk quote: "It has all the requested features, including columns for the date of service, work performed, mileage at service, and cost." https://www.vertex42.com/ExcelTemplates/vehicle-maintenance-log.html

### 5) Add uncertainty flags in your log

Use a note field like:
- "Spec confirmed from cap only"
- "Forum recommendation, pending FSM verification"
- "Mixed-source conflict"

### 6) Re-validate after major changes

If you change transmission, differential, cooling hardware, or add forced induction/track use, re-check fluid choices and intervals.

> Community confirmation quote: "Just bought a NB, looking to do some maintenance" https://duckduckgo.com/html/?q=site%3Aforum.miata.net+comprehensive+list+maintenance+NB

## Baseline Reference Table (NB1/NB2)

| System | What to Record | Practical NB Note |
|---|---|---|
| Engine oil | Viscosity, API/ACEA level, fill amount | Varies by climate/use; manual-first |
| Manual transmission | Fluid type/spec and fill amount | Gearbox-specific behavior (shift feel) can vary by fluid |
| Differential | Gear oil type/grade and quantity | LSD/open setup can affect fluid choice |
| Coolant | Chemistry compatibility + concentration | Never mix unknown coolant types without confirmation |
| Brake/Clutch hydraulic | DOT type and flush interval | Use cap/manual spec; track use shortens interval |
| Power steering | Fluid type wording from manual/cap | Some NB references cite ATF-equivalent language |
| Consumables | Filters, washers, belts, plugs | Track part numbers per service entry |

## Torque Specs / Capacities

- This appendix intentionally avoids publishing single universal NB capacities/torques.
- Use year/VIN-specific workshop data and system-specific guides in this manual for final numbers.

## Verification / Post-service checks

- Every fluid service entry includes: date, odometer, exact fluid type, quantity used, and next due.
- Any uncertainty/conflict is explicitly marked in notes.
- No “unknown fluid currently installed” entries remain for critical systems.

## Uncertainty / Open Questions

- Publicly available sources in this run support the *logging and verification process* strongly, but not one complete, extractable NB-only official fluid/capacity master table.
- Community threads are useful for baseline planning but should not override VIN/year workshop references where values conflict.
