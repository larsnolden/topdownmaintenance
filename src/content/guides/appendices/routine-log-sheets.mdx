## Before You Start / Safety

- A log sheet is not a replacement for safety inspection or workshop diagnosis.
- Record work **immediately after completion** to avoid missed details.
- Keep receipts/photos with each entry for traceability.

> Warning: "Consistency matters more than complexity. A simple spreadsheet updated regularly outperforms an advanced system used sporadically." https://autosist.com/blog/free-vehicle-maintenance-log-template-excel-download/

## Required Tools

- Notebook, spreadsheet, or printable PDF log
- Odometer reading (or trip logger)
- Phone camera for invoices/part labels
- Folder/cloud storage for evidence

## Required Parts / Fluids

- Not applicable (logging-only appendix)

## Model-specific notes (NB1 vs NB2)

- Keep one log per car and include exact model/year (NB1 1998–2000, NB2 2001–2005 market-dependent naming).
- NB cars often have mixed aftermarket parts; record brand/part number every time.

> Manual-reference quote: "1999-2001 Mazda Miata Service Repair Manual ... 2005 Mazda Miata Service Repair Manual" https://www.mellens.net/mazda/

## Step-by-Step Procedure

### 1) Set up your core log fields once

Minimum columns:
- Date
- Odometer (km/mi)
- Service performed
- Parts/fluids used
- Quantity/spec
- Cost
- Next due (date + odometer)
- Notes/issues found

> High-risk quote: "It has all the requested features, including columns for the date of service, work performed, mileage at service, and cost." https://www.vertex42.com/ExcelTemplates/vehicle-maintenance-log.html

### 2) Add NB-specific fields

Recommended extra columns for MX-5 NB:
- Rust check result (sills/arches/underbody)
- Drain status (clear/slow/blocked)
- Top condition notes (if soft-top)
- Alignment/tire wear notes

### 3) Use one row per completed action

Do not batch vague notes like “service done.”

Good: “Engine oil + filter, 10W-40 API SL, 3.8 qt, filter Mazda B6Y1-14-302A-equivalent, no leaks.”

### 4) Link proof to every major entry

Attach invoice number, photo, or worksheet reference.

> High-risk quote: "I made some cosmetic changes, added an invoice column for recording an invoice or receipt #..." https://www.vertex42.com/ExcelTemplates/vehicle-maintenance-log.html

### 5) Add a recurring review checkpoint

Review log monthly (or every 1,000 km / 600 mi) to spot overdue items.

### 6) Track baseline and interval work distinctly

Tag entries as:
- **Baseline** (newly purchased car catch-up)
- **Interval** (scheduled repeat service)
- **Corrective** (fault-related repair)

> Community confirmation quote: "Google sheets Maintenance Logs" https://www.reddit.com/r/Miata/search/?q=maintenance+log+NB&restrict_sr=1

## Torque Specs / Capacities

- This appendix does not provide torque/capacity values.
- Pull those values from the relevant NB service guide and copy them into your log entry notes when work is completed.

## Verification / Post-service checks

- Confirm each performed service has: date, odometer, parts/fluid details, and next-due target.
- Confirm major services include evidence (invoice/photo).
- Confirm there are no “unknown last service” items remaining.

## Uncertainty / Open Questions

- Publicly accessible sources in this run strongly support structured maintenance logging, but there is no single Mazda NB factory “owner log format” extract available in this environment.
- Use this appendix template as a practical framework and adapt fields to your climate/use-case (track days, winter salt exposure, short-trip duty).
