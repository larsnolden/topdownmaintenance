## Before You Start / Safety

- Switch ignition OFF before pulling fuses.
- Use the correct fuse puller/pliers and avoid shorting adjacent terminals.
- Replace only with the exact same fuse type and amperage.

> Warning: "Replace failed fuses with proper amp rating, not doing so can result in electrical system damage." https://www.2carpros.com/articles/how-to-check-a-car-fuse

## Required Tools

- Fuse puller or small insulated pliers
- Flashlight
- Multimeter or test light (recommended)
- Correct replacement fuses (same style/rating)

## Required Parts / Fluids

- Replacement blade fuses in matching amperage values

## Model-specific notes (NB1 vs NB2)

- Fuse layout and assignment can vary by NB year/market and by whether covers/labels were replaced.
- Always verify location using your fuse-box legend/manual, not internet images alone.

> Manual-reference quote: "Welcome to Mazda Miata Factory Service Manuals" https://www.mellens.net/mazda/

> Community quote: "My fuse box looks nothing like any of the images I can find online." https://www.reddit.com/r/Miata/search/?q=NB+fuse+box&restrict_sr=1

## Step-by-Step Procedure

### 1) Confirm the failed circuit symptom

Identify what stopped working (e.g., horn, lights, accessory). Check whether related systems also fail, which may indicate a shared supply issue.

### 2) Locate the correct fuse box and legend

Use the cover legend and year-specific documentation to identify the suspect fuse(s). NB cars may have multiple fuse locations.

### 3) Inspect fuse visually first

Remove the fuse and check the element for a break/charred mark. Compare with a known-good fuse if uncertain.

### 4) Confirm power side and load side (recommended)

With ignition state appropriate to the circuit, verify fuse terminal power with a test light/meter.

> High-risk quote: "A fuse with power on just one side is blown and needs replacement." https://www.2carpros.com/articles/how-to-check-a-car-fuse

### 5) Replace with exact matching amperage

Install only the same rating and type. Never up-rate a fuse to “stop it blowing.”

> High-risk quote: "Never use a fuse with a higher amperage than the one that blew... can cause electrical damage or a fire." https://www.wikihow.com/Check-Fuses

### 6) Re-test and check for repeat failure

If the replacement blows again, stop replacing repeatedly and diagnose downstream short/load fault.

> Community confirmation quote: "Starting problem, main fuse blown. Will it happen again?" https://www.reddit.com/r/Miata/search/?q=NB+blown+fuse&restrict_sr=1

## Torque Specs / Capacities

## Verification / Post-service checks

- Confirm affected circuit works normally.
- Confirm no secondary warning lights/codes appear.
- Recheck fuse temperature after brief operation (should not overheat).

## Uncertainty / Open Questions

- Public sources in this run agree strongly on process controls (correct ID, same-value replacement, and no overfusing).
- A single extractable NB1/NB2 all-market fuse-map table was not reliably available in this environment; final slot identification should be confirmed against year-specific labels/manual pages.
