## Before You Start / Safety

This guide is for **Mazda MX-5 NB (1998-2005)** only.

Headlight bulb swaps are simple, but incorrect bulb handling/fitment can shorten bulb life or damage connectors/housings.

> Warning: Ensure the vehicle is off and the headlamp assembly has cooled before bulb handling to reduce burn risk. https://www.wikihow.com/Replace-a-Headlight-Bulb

> High-risk quote: Do not touch halogen bulb glass with bare fingers; oils can create hot spots and early failure. https://www.wikihow.com/Replace-a-Headlight-Bulb

> Manual-reference quote: Mellens indexes NB-era manuals (1999-2001 and 2005), so exact bulb type and any trim-specific access steps should be confirmed by year/VIN before purchase and installation. https://www.mellens.net/mazda/index.html

## Required Tools

- Clean nitrile gloves
- Small flat screwdriver/plastic trim tool (if cover clip removal is needed)
- Lint-free cloth and isopropyl alcohol (for accidental glass contact cleanup)

## Required Parts / Fluids

- Correct replacement headlight bulb type for your NB year/market
- Optional replacement connector/pigtail if existing connector is heat-damaged

### Practical bulb-selection rules

- Match the exact bulb code printed on the old bulb or confirmed by year/VIN documentation.
- Match wattage to OEM specification (avoid higher-wattage “off-road” bulbs in stock housings/connectors).
- Replace left/right as a pair for consistent color and output.

## Model-specific notes (NB1 vs NB2)

### NB1 (1998-2000)
- Verify actual bulb code from existing bulb or FSM/owner’s documentation before ordering.

### NB2 (2001-2005)
- Same caution: aftermarket headlight assemblies/conversions can use non-stock bulb types.

> Community quote: r/Miata search for NB headlight bulb replacement shows repeated discussions around bulb recommendations and retrofit compatibility on NB1/NB2 housings. https://www.reddit.com/r/Miata/search/?q=nb%20headlight%20bulb%20replacement&restrict_sr=1

## Step-by-Step Procedure

### 1) Confirm bulb type and side-to-side strategy

- Check existing bulb marking and cross-check with year/VIN documentation.
- Replace in pairs when possible to keep brightness/color matched.

### 2) Access rear of headlight housing

- Open hood.
- Stand facing the headlight from engine bay side and locate the rear bulb area.
- Remove rear dust cap/connector retainer as needed (rubber cap or plastic cover depending on housing).
- Inspect for moisture ingress signs in housing while open (water marks, fogging residue, rusty clip hardware).

### 3) Remove old bulb carefully

- Unplug connector straight back.
- Release spring clip/retainer ring.
- Remove bulb without twisting contacts or damaging seal seat.

### 4) Inspect connector and wiring

- Check for melted plastic, darkened terminals, brittle insulation.
- If connector is heat-damaged, repair/replace connector before fitting new bulb.

> Community confirmation quote: old.reddit NB headlight search context includes "melted headlight bulb connector in 1998 NB" style reports, supporting connector inspection during bulb replacement. https://old.reddit.com/r/Miata/search?q=nb+headlight+bulb+replacement&restrict_sr=on

### 5) Install new bulb correctly

- Handle bulb by base only.
- Seat bulb tabs fully in correct orientation (bulb should sit flush, not cocked).
- Re-secure retainer and reconnect connector firmly.
- Refit dust cap/seals to prevent moisture entry.
- If glass was touched accidentally, clean with isopropyl alcohol and lint-free cloth before installation.

### 6) Function and pattern check

- Test low/high beam on both sides.
- Confirm beam color/brightness match.
- Do a quick wall-pattern sanity check; perform formal aim adjustment if disturbed.

## Torque Specs / Capacities (if applicable)

No meaningful torque specs are typically required for standard headlight bulb replacement.

## Verification / Post-service checks

- Both low and high beams function consistently
- No flicker under vibration
- No connector heating smell after several minutes of operation
- Dust cap/seals fully seated with no condensation development

## Uncertainty / Open Questions

- Exact bulb code and rear-cap access details can vary by market/year and aftermarket headlamp conversions.
- Therefore exact bulb code and any market-specific access differences are explicitly deferred to year/VIN documentation.
