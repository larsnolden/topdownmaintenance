## Before You Start / Safety

This guide is for **Mazda MX-5 NB (1998-2005)** only.

Exterior bulb replacement is simple, but wrong bulb type, poor socket contact, or moisture intrusion can cause repeat failures.

> Warning: Switch lamps off and let bulbs cool before handling to avoid burns and cracked hot glass. https://www.wikihow.com/Replace-a-Headlight-Bulb

> High-risk quote: Handle bulbs by the base and avoid contaminating bulb glass/contact surfaces. https://www.wikihow.com/Replace-a-Tail-Light-Bulb

> Manual-reference quote: Mellens indexes NB-era manuals (1999-2001 and 2005), so exact bulb map and housing access method should be confirmed by year/VIN before ordering parts. https://www.mellens.net/mazda/index.html

## Required Tools

- Small screwdriver or trim tool (for covers/clips)
- Gloves
- Contact cleaner (optional)
- Lint-free cloth

## Required Parts / Fluids

- Correct replacement bulbs for each function (tail/brake/turn/license)
- Optional replacement socket/pigtail if terminals are heat-damaged or corroded

### Practical bulb-fitment rules

- Match exact base type and filament layout from old bulb/housing marking.
- For dual-function circuits (for example tail+brake), use correct dual-filament equivalent where required.
- Avoid mixing LED and incandescent on one side unless load/flash strategy is handled correctly (hyperflash risk).

## Model-specific notes (NB1 vs NB2)

### NB1 (1998-2000)
- Tail lamp internals and lens styles can differ from NB2; verify socket fit and bulb type from existing lamp or manual.

### NB2 (2001-2005)
- Same caution: aftermarket lamp assemblies are common and may use different bulb bases/wattage.

> Community quote: r/Miata search for NB tail-light bulb issues includes repeated reports of fitment differences and mixed OEM/aftermarket housings. https://www.reddit.com/r/Miata/search/?q=nb%20tail%20light%20bulb&restrict_sr=1

## Step-by-Step Procedure

### 1) Confirm failed function and bulb type

Identify exactly which function failed:
- tail/running,
- brake,
- turn,
- reverse,
- license plate.

Do not assume one bulb handles all functions on all assemblies.

### 2) Access lamp housing

- Open trunk/hatch or relevant access panel.
- Locate rear of target lamp assembly from inside trunk/boot trim.
- Remove socket holder or rear cover as required.
- Inspect gasket and housing for moisture signs while open.

What moisture evidence looks like:
- water droplets/fog film inside lens,
- white mineral residue trails,
- rusty fasteners or greened terminals.

### 3) Remove old bulb and inspect socket

- Twist/pull bulb per socket design.
- Check for green corrosion, heat-darkened plastic, or weak terminal tension.

If socket is damaged, replace socket/pigtail before installing new bulb.

### 4) Install correct bulb and reassemble

- Match base type and function.
- Seat bulb fully; verify positive lock in socket.
- Refit seals/covers to prevent moisture ingress.

### 5) Functional checks

- Test each exterior function with helper or reflection method.
- Confirm turn-signal flash rate is normal and left/right cadence matches.
- Confirm brake and tail intensity difference is correct.
- Confirm license plate lamps are both working and similarly bright.

### 6) If problems persist

- Recheck fuses and shared-ground condition.
- Recheck connector corrosion and terminal tension.
- Inspect housing for water entry paths.

Quick diagnosis hint:
- one side dim or feedback between functions often points to ground/contact issue,
- both sides out on one function often points to fuse/supply issue.

## Torque Specs / Capacities (if applicable)

No meaningful torque specs apply for routine exterior bulb swaps.

## Verification / Post-service checks

- All rear/exterior lamp functions work correctly
- No rapid hyperflash (unless expected by LED conversion setup)
- No flicker when tapping housing/lightly closing trunk
- No moisture buildup after rain/wash

## Uncertainty / Open Questions

- Exact bulb code per function can vary by market/year and aftermarket lamp assemblies.
- Therefore exact bulb code per function should be verified by year/VIN manual and existing housing variant.
