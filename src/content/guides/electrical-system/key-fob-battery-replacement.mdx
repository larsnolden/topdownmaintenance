## Before You Start / Safety

This guide is for **Mazda MX-5 NB (1998-2005)** only.

NB key/remote hardware varies by year/market and prior ownership changes. Confirm your fob style and battery type before opening the case.

> Warning: Coin-cell batteries are hazardous if swallowed and must be kept away from children/pets. https://www.energizer.com/batteries/coin-cell-batteries

> High-risk quote: Open the fob case gently with a non-marring tool to avoid cracking tabs or damaging the PCB/buttons. https://www.wikihow.com/Change-the-Battery-in-a-Key-Fob

> Manual-reference quote: Mellens indexes NB-era manuals (1999-2001 and 2005), so remote/keyless system details and programming differences should be confirmed by exact year/VIN when troubleshooting post-swap issues. https://www.mellens.net/mazda/index.html

## Required Tools

- Small flat plastic pry tool (or plastic pick)
- Lint-free cloth
- Correct coin-cell battery (verify exact code from old cell)

## Required Parts / Fluids

- Replacement coin-cell battery matching original spec (same code/chemistry)

### Practical battery-code guidance

Most remotes use a 3V lithium coin cell (commonly **CR20xx** family such as CR2025 or CR2032), but thickness differs and must match exactly.

Rules:
1. Read the code printed on the old cell.
2. Match full code (for example, CR2025 ≠ CR2032).
3. Use a fresh, reputable-brand cell in original packaging.

## Model-specific notes (NB1 vs NB2)

### NB1 (1998-2000)
- Remote/keyless hardware may differ by market/trim; some cars may not have OEM remote hardware.

### NB2 (2001-2005)
- Multiple fob shells and aftermarket replacements exist; verify case style and battery orientation directly.

## Step-by-Step Procedure

### 1) Confirm battery type before opening

Read the old battery code first (if visible) and buy identical replacement.

Do not rely on “looks similar” coin cells.

### 2) Open the fob carefully

- Remove physical key insert if applicable.
- Locate the seam/notch (often at one end of the shell).
- Pry at seam/notch with plastic tool.
- Twist gently just enough to release first clip, then work around shell.
- Separate shell halves slowly to protect clips and button pad.

### 3) Note battery orientation

Before removing old cell, note **+/- orientation** and contact position.

Take a quick phone photo for reference.

### 4) Replace battery

- Remove old cell gently.
- Clean contacts lightly if needed.
- Install new cell in exact same orientation.

### 5) Reassemble and function check

- Re-seat button pad and PCB correctly.
- Snap case closed evenly.
- Test lock/unlock/trunk functions near vehicle.

### 6) If remote still fails

- Recheck battery orientation and contact tension.
- Confirm battery is fresh and correct type.
- Inspect for cracked solder joints or worn button switches.
- Follow year/VIN-specific programming/registration steps if required.

Quick triage:
- LED on fob lights but car does not respond => likely pairing/vehicle-side issue.
- No LED/no response from fob => likely battery/contact or fob PCB/button issue.

## Torque Specs / Capacities (if applicable)

No torque specs apply for typical fob battery replacement.

## Verification / Post-service checks

- Lock/unlock works at normal range
- Trunk release button (if equipped) works
- Fob case fully clipped with no rattles
- No intermittent button response

## Uncertainty / Open Questions

- NB remote/fob hardware can vary significantly by market, trim, and aftermarket alarm/keyless retrofits.
- Therefore exact battery code and any reprogramming sequence should be verified by your specific fob and year/VIN documentation.
