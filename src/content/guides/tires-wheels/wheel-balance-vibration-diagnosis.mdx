## Before You Start / Safety

This guide is for **Mazda MX-5 NB (1998-2005)** only.

Speed-dependent vibration is often blamed on wheel balance, but root causes can include tire defects, bent wheels, wheel-bearing play, alignment issues, or driveline faults.

> Warning: Perform test drives only on safe roads and legal speeds; do not exceed speed limits to “chase” vibration. https://www.tirerack.com/upgrade-garage/how-to-diagnose-a-vibration-issue

> High-risk quote: Vibration diagnosis should isolate whether the disturbance comes from tire/wheel assembly, vehicle speed range, and steering input response before replacing parts. https://www.tirerack.com/upgrade-garage/how-to-diagnose-a-vibration-issue

> Manual-reference quote: Mellens indexes NB-era manuals (1999-2001 and 2005), so wheel/tire specs, runout checks, and chassis torque data should be confirmed by exact year/VIN. https://www.mellens.net/mazda/index.html

## Required Tools

- Tire pressure gauge
- Tread depth gauge
- Flashlight
- Jack + stands (for play/runout checks)
- Access to a modern wheel balancer (road-force capability preferred)

## Required Parts / Fluids

- None for diagnosis-only work
- Wheel weights, tire service, or wheel repair/replacement as indicated by test results

## Model-specific notes (NB1 vs NB2)

### NB1 (1998-2000)
- Many cars run non-stock wheel/tire sizes; baseline assumptions from OEM setup may not apply.

### NB2 (2001-2005)
- Same caution: mixed wheel offsets, lowered ride height, and bushing wear can amplify otherwise minor imbalance.

> Community quote: r/Miata search results for NB high-speed vibration repeatedly show mixed causes (balance, bent wheel, alignment, bearings) rather than one universal fix. https://www.reddit.com/r/Miata/search/?q=nb%20vibration%20at%20speed&restrict_sr=1

## Step-by-Step Procedure

### 1) Characterize vibration precisely

Record:
- speed range where vibration starts/peaks,
- whether felt in steering wheel, seat, or whole body,
- whether braking/turning/load changes it.

### 2) Basic tire checks first

- Set cold tire pressures to correct baseline.
- Inspect for uneven wear, cupping, bulges, flat spots.
- Check lug torque and obvious wheel damage.

### 3) Rotate wheel positions as a diagnostic step

If vibration location changes after front/rear rotation, tire/wheel assembly is strongly implicated.

### 4) Perform balance check (road force preferred)

Use dynamic balancing, and if available, road-force measurement for better diagnosis of tire/wheel uniformity issues.

> High-risk quote: Road-force balancing can reveal uniformity problems a basic spin balance may miss. https://www.hunter.com/media-center/road-force-balancing-explained/

### 5) Rule out look-alike faults

If vibration persists after proper balance:
- inspect wheel bearing play/roughness,
- inspect bent wheel runout,
- inspect suspension joints/bushings,
- inspect driveline sources if vibration is load-dependent.

### 6) Re-test and verify repair path

After each corrective action, re-test on same route/speed band to confirm change.

> Community confirmation quote: old.reddit NB vibration search context shows many owners fixing only part of the issue until they combined balancing with alignment/suspension checks. https://old.reddit.com/r/Miata/search?q=nb+vibration+at+speed&restrict_sr=on

## Torque Specs / Capacities (if applicable)

- Wheel lug torque: follow NB year/VIN specification (see FSM and wheel-lug guide).

## Verification / Post-service checks

- Vibration reduced/eliminated in previously affected speed range
- Steering wheel no longer oscillates abnormally
- No abnormal tire wear progression after follow-up checks

## Uncertainty / Open Questions

- Exact vibration thresholds and acceptable road-force values depend on wheel/tire setup and shop equipment.
- Community evidence in this run was partly search/snippet level.
- Therefore final pass/fail values are intentionally conservative and deferred to qualified measurement + year/VIN specs.
