## Before You Start / Safety

- Tire pressures must be set from **cold** values (car parked several hours or driven very short distance only).
- Do not use sidewall max-pressure text as your operating target.
- Refit valve caps after adjustment to keep dirt/moisture out of valve cores.

## Required Tools

- Reliable pressure gauge (digital preferred)
- Air source (compressor/pump)
- Valve-cap tool/spare caps (optional)
- Notebook/phone to record FL/FR/RL/RR readings

Tip: keep one known-good gauge in the glovebox and use it consistently. Mixed gauges can differ by several psi and create false troubleshooting signals.

## Where to find the correct target on NB

Use this order of priority:
1. Driver-door-jamb pressure label (primary)
2. Owner manual pressure table for your wheel/tire size
3. Tire-shop setup sheet only if it matches (1) or (2)

NB cars often run multiple wheel sizes depending on market/options, so one internet number is not sufficient.

## Step-by-Step Procedure

### 1) Measure all four tires cold

Remove each valve cap and press gauge squarely on valve stem for a clean seal.
Record each corner before adding/releasing air.

If you hear a continuous hiss while measuring, reseat the gauge and measure again—bad gauge angle can under-read pressure.

### 2) Compare against your label target

Adjust each tire to its specified front/rear cold value.

- Add air in short bursts.
- Recheck after each burst.
- If overfilled, bleed briefly and remeasure.
- Keep left/right pressures equal on the same axle unless the factory label specifies otherwise.

### 3) Include spare (if fitted)

If your NB has a temporary spare, check and maintain it separately so it is usable when needed.

### 4) Hot-tire fallback rule (if you cannot wait)

If tires are warm from driving, set a temporary value and then **recheck cold later**.
Industry guidance commonly uses about **+4 psi** as a temporary warm adjustment reference before final cold correction.

Example: if door label says 29 psi cold and tires are hot, set around 33 psi temporarily, then reset to 29 psi at next true cold check.

### 5) Leak triage if one corner drops repeatedly

If one tire repeatedly loses pressure faster than others, inspect:
- puncture in tread,
- valve core leakage,
- cracked valve stem,
- bead/rim corrosion leak.

Do not just top up repeatedly without finding the leak.

## Practical pressure-management notes

- Seasonal temperature swings change measured pressure significantly; check monthly and before long trips.
- Rule of thumb: tire pressure changes roughly 1 psi per ~10°F (about 6°C) ambient change.
- Aim for consistency side-to-side on each axle unless your label specifies different values.
- If steering pulls after pressure correction, inspect alignment/tires rather than adding random PSI offsets.

## Verification / Post-service checks

- All four tires match target values when cold
- Valve caps reinstalled
- No TPMS-style warning behavior (if retrofit/aftermarket system present)
- Straight tracking and even steering feel on short road check
